# Práctica 1: Creación de mi primera página web con Reflex

## Descripción

Este proyecto consiste en la creación de una página web básica utilizando Reflex, siguiendo la documentación oficial del framework. La aplicación está basada en el ejemplo del contador interactivo, donde se utilizan conceptos como State, event handlers y componentes visuales.

La página incluye un título principal, un texto de bienvenida y un contador interactivo con botones para incrementar, decrementar y reiniciar el valor.

## Tecnologías utilizadas

- Python
- Reflex
- uv
- Git
- GitHub

## Requisitos

Antes de ejecutar el proyecto, se necesita tener instalado:
